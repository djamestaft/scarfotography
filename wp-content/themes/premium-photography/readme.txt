== Theme Description ==
Premium Photography is an elegant, simple, clean, modern, minimal and fully responsive Twitter bootstrap based premium quality WordPress theme which is mainly designed for photographers for showcasing their portfolios or writing their experiences though the blog-posts but can be used for travel, corporate, portfolio, technology tips, personal and any other creative and minimalistic style website.Premium Photography theme has a custom widget which displays latest posts' featured images. Theme comes with 'Theme Options' through which you can change Logo, Favicon, Social media links, Copyright Text etc.

== Features ==
Responsive design(Bootstrap), Theme Options, Custom Menu, supports social media, Right-Sidebar, Left-Sidebar, Full-Width.

== Tags ==
Right-sidebar to set the layout of overall theme.


== Installation ==

1. Primary:
 = Login to your wp-admin account and go to Appearance -> Themes.
 = Select "Install" tab and click on the "Upload" link.
 = Select "theme .zip" and click on "Install Now" button.
 = In case of errors, use an alternate method.

2. Alternate:
 = Unzip the template file (premiumphotography.zip) that you have downloaded.
 = Upload the entire folder (premiumphotography) to your server via FTP and place it in the /wp-content/themes/ folder.
 = Do not change the directory name.
 = The template files should be there now: /wp-content/themes/premiumphotography/index.php (example).
3. Log into your WP admin panel and click on "Appearance". Go to "Themes" tab.
4. Now click on "premiumphotography" to activate the theme.
5. Complete all of the required inputs in the theme Options page (in the WP admin panel) and click "Save".

6. Basic Setting:
 = Login to your wp-admin area and go to Appearance -> Theme Options.
 = Select Basic Setting Tab.
 = Upload the logo, Favicon and you can add copyright text from here.

7. Social Media:
 = Login to your wp-admin area and go to Appearance -> Theme Options.
 = Select Social Options tab
 = Here you can add your Facebook, Twitter, Linkedin, Google Plus.
